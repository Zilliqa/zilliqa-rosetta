# zilliqa-rosetta
Zilliqa node which follows Rosetta Blockchain Standard

## Build docker image

```shell script
sh ./build_docker.sh
```

## Running docker image

## How to use

### configuration

The default configuration file is config.local.yaml

```shell script
Host: "0.0.0.0"
Port: 8080
```

* Host : rosetta restful api host
* Port : resetta restful api port
